# Paeony

一个高性能bittorrent 追踪器，使用C++11进行编写。支持多种BT客户端使用IPV4/IPv6接入。

## 新的功能
-[ ] 双栈接入
-[ ] 支持缓存
-[ ] 支持多种客户端白名单
-[ ] 支持不同种子free模式，包括2xfree,free,50%free,30%free
